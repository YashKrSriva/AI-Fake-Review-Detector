# AI Fake Review Detector

A modern AI-powered web application that detects whether a product review is **Fake** or **Genuine** using **Machine Learning**, **Natural Language Processing (NLP)**, **React**, and **Flask**.

---

# 🚀 Features

* AI-powered fake review detection
* Modern React + Vite frontend
* Flask backend API
* Machine Learning model using Scikit-learn
* Responsive glassmorphism UI
* Real-time review prediction
* REST API integration using Axios
* Fast and interactive user experience

---

# 🛠️ Technologies Used

## Frontend

* React.js

* CSS

## Backend

* Python
* Flask

## Machine Learning

* Scikit-learn
* Pandas
